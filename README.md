## 🚀 Инструкция по запуску

### 📍 Запуск Linux MacOS

```bash
  git clone https://github.com/codi-mvp/weather-bot-tg
```
```bash
  cd weather-bot-tg
```
```bash
  npm install
```
```bash
  npm run test
```

### 📍 Запуск Windows

Скачайте исходный код последней версии, распакуйте его в папку,
откройте терминал в папке с проектом и выполните:
```bash
  npm install
```
```bash
  npm run test
```
